# Hospital Readmission Prediction

Machine Learning project that predicts whether a patient is likely to be readmitted to the hospital.

## Features
- Data preprocessing
- Exploratory Data Analysis
- Machine Learning model
- Streamlit web application

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Streamlit

pip install -r requirements.txt
